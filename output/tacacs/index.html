TACACS+

TACACS+ is a protocol for remote authentication and access control in network security.

Description

TACACS+ (Terminal Access Controller Access-Control System Plus) is a protocol developed by Cisco for providing secure authentication, authorization, and accounting (AAA) services for users who access network resources. It operates over TCP and encrypts the entire payload of the communication, ensuring that sensitive information such as passwords is not transmitted in plain text. TACACS+ separates the functions of authentication, authorization, and accounting, allowing for more granular control over user permissions and activities. This separation makes it easier for administrators to manage user access and track usage across the network. TACACS+ is commonly used in environments with a need for centralized control over user access to devices such as routers, switches, and firewalls. Its flexibility and security features make it a popular choice for enterprise networks that require robust security measures to protect against unauthorized access.

Examples

  • Used by network administrators to control access to routers and switches.
  • Enables logging and monitoring of user activities for compliance and auditing purposes.

Additional Information

  • TACACS+ is an enhancement of the original TACACS protocol, providing better security and functionality.
  • It supports multiple authentication methods, including Kerberos and RADIUS integration.
