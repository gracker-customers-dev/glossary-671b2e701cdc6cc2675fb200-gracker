Fully-Qualified Domain Name (FQDN)

A Fully-Qualified Domain Name (FQDN) is the complete domain name for a specific computer or host on the internet.

Description

A Fully-Qualified Domain Name (FQDN) uniquely identifies a host within the Domain Name System (DNS) hierarchy. It consists of two main parts: the hostname and the domain name. The hostname is the specific name assigned to a device, while the domain name includes the top-level domain (TLD) and any intermediate domain names. An FQDN provides an absolute address that can be resolved to an IP address, enabling users and applications to locate resources on the internet or local networks. FQDNs are essential in network security as they help in identifying and authenticating web servers and services. They are used in various security protocols, such as SSL/TLS, where the FQDN is matched against the certificate presented by the server to ensure that the connection is secure. Proper management of FQDNs is crucial to prevent attacks like DNS spoofing and ensure that users are connecting to the intended servers.

Examples

  • www.example.com
  • mail.google.com

Additional Information

  • An FQDN always ends with a dot (.) indicating the root of the DNS hierarchy, though this is often omitted in everyday use.
  • FQDNs are case insensitive, meaning that 'Example.com' and 'example.com' refer to the same domain.
