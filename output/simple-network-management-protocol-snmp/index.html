Simple Network Management Protocol (SNMP)

A protocol used for network management and monitoring of network devices.

Description

Simple Network Management Protocol (SNMP) is a standard protocol used to manage and monitor network devices such as routers, switches, servers, and printers. It operates in a client-server model where the network management system (NMS) acts as the manager and the devices being monitored act as agents. SNMP allows for the collection of device status and performance metrics, configuration management, and alerts for operational failures. It utilizes a set of operations like GET, SET, and TRAP to interact with devices. SNMP employs a hierarchical structure for identifying managed objects using Object Identifiers (OIDs). Security has been a concern with SNMP versions 1 and 2, as they use community strings for authentication, which are sent in plain text. SNMP version 3 was introduced to address these vulnerabilities, providing enhanced security features such as authentication, encryption, and access control. Through SNMP, network administrators can ensure the reliability and performance of networks, making it a critical tool in network management.

Examples

  • Monitoring bandwidth usage on a router using SNMP queries.
  • Receiving alerts about device failures through SNMP traps.

Additional Information

  • SNMP operates over UDP, typically using port 161 for queries and port 162 for traps.
  • It is widely supported by various network devices and management software.
