Boot Record Infector

A type of computer virus that infects the boot sector of storage devices.

Description

A Boot Record Infector is a malicious software program that targets the boot sector of a storage device, such as a hard drive or USB drive. The boot sector is the part of the storage that contains the code needed to start the operating system. When a computer is powered on, the BIOS reads the boot sector and executes the code therein. If a boot record infector is present, it can execute its payload before the operating system loads, potentially compromising the entire system. These types of viruses can spread from one system to another through infected removable media, such as floppy disks or USB drives. Once a system is infected, the boot record infector can manipulate system files, create backdoors for remote access, or even render the operating system unusable. Due to the low-level nature of boot sector infections, they can be more challenging to detect and remove compared to other types of malware. Prevention strategies include using up-to-date antivirus software and ensuring that systems boot from secure and trusted sources.

Examples

  • Stoned Virus: One of the earliest boot sector viruses that infected floppy disks.
  • Michelangelo Virus: A well-known boot sector virus that emerged in the early 1990s and activated on March 6.

Additional Information

  • Boot record infectors can be particularly dangerous because they can evade detection by traditional antivirus software until the boot process begins.
  • To remove a boot sector virus, specialized tools or recovery methods may be required, as simply deleting infected files may not suffice.
